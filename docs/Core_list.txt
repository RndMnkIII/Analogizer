# JT Config File
Uses Pupdate utility to create `crtcfg.bin` file stored in `Assets/jtpatreon/common` folder. The settings stored in this file are shared between all JT core.

# Analogizer File
Uses Pupdate utility to create `analogizer.bin` file stored in `Assets/analogizer/common` folder. The settings storeD in this file are shared between all standard cores (non-JT).

# Pocket Menu
Use the Pocket's own menu to display and select options. This is done for each core.
